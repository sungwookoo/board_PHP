
- 파일을 첨부한 게시글 수정 시 
다른 파일을 첨부 하지 않으면 기존 파일 그대로 유지
다른 파일을 첨부하면 새로 첨부한 파일로 첨부파일 수정
- 게시글 보기에서 첨부파일 클릭 시 chrome에서는 클릭 시 즉시 다운로드 됨
-> iexplorer에서는 클릭 시 다운로드 하지 않고 바로 실행됨 
-> 파일명 마우스 우클릭 시 다른 이름으로 대상 저장으로 다운로드 가능
- 첨부된 파일이 있다면 첨부파일명을 표시, 없다면 '첨부된 파일이 없습니다.' 표시


조회수 새로고침 중복 증가 방지 로직 아이디어
게시글 클릭 -> 인덱스번호만 따서 hit +1 업데이트하는 php 페이지로 이동
-> hit+1업데이트가 끝나면 바로 /read.php?idx=(해당글 인덱스번호) 로 이동
